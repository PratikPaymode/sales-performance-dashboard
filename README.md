## 📊 E-Commerce Sales Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **E-Commerce Sales Dashboard** built using Power BI to analyze key business metrics such as revenue, profit, quantity sold, and sales distribution across regions, categories, and payment modes.

---

## 🛠️ Tools & Technologies

* Power BI (Data Visualization)
* Basic Data Transformation (Power BI Query Editor)

---

## 📁 Dataset

* Combined dataset created by merging multiple tables
* Includes fields such as Sales, Profit, Quantity, State, Category, and Payment Mode
* Dataset file: `dataset comp.csv`

---

## 📈 Key Metrics (KPIs)

* Total Sales Amount: 438K
* Total Profit: 37K
* Total Quantity Sold: 5615
* Average Order Value (AOV): 121K

---

## 📊 Dashboard Features

* Sales and profit analysis by state
* Category and sub-category performance
* Monthly profit trends
* Payment mode distribution
* Interactive filters (Quarter, State)

---

## 🔍 Key Insights

* Clothing category contributes the highest quantity sold
* Maharashtra generates the highest revenue
* Profit varies across months showing seasonal trends
* COD is the most preferred payment method
